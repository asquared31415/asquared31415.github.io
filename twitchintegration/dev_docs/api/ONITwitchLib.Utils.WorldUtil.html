CreateWorldWithTemplate(GameObject, Vector2I, string, Action<WorldContainer>)

Allocates Grid space for a world and creates the world with a template, calling the callback once the template is placed.

[PublicAPI]
[CanBeNull]
public static WorldContainer CreateWorldWithTemplate(GameObject worldGo, Vector2I size, string template, Action<WorldContainer> callback = null)

Parameters

worldGo GameObject

The object that is an instance of a ClusterGridEntity to set up as the world.

size Vector2I

The size of the world to create, in cells.

template string

The template to place in the world.

callback Action<WorldContainer>

If present, the callback to call after placing the template.

Returns

WorldContainer

The WorldContainer for the newly created world.

Remarks

Important notes for modder usage:
The WorldContainer does not have an overworldCell set. You should set that field and add the cell to SaveLoader.Instance.clusterDetailSave.overworldCells.
None of the Grid is revealed. Use GridVisibility.Reveal or Grid.Reveal to reveal the grid.
