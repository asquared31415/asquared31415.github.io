Parameters

minAngleInclusive float

The lower bound of the angle of the output vector in degrees, inclusive.

maxAngleExclusive float

The upper bound of the angle of the output vector in degrees, exclusive.

Returns

Vector2

A vector with length one that has an angle between minAngleInclusive and maxAngleExclusive.

Remarks

Note that 0 degrees represents the vector (1, 0), and the angle increases counter-clockwise, such that 90 degrees is (0, 1).
