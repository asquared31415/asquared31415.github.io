CreateRuntimeTypeActionDelegate(MethodInfo, object, Type)

Creates a Action<T> delegate for a method with its generic types replaced with object, with the real type determined at runtime.

[PublicAPI]
public static Action<object> CreateRuntimeTypeActionDelegate(MethodInfo methodInfo, object arg0, Type arg1Type)

Parameters

methodInfo MethodInfo

The method to call.

arg0 object

The optional this object for the delegate.

arg1Type Type

The Type of the first argument to the method.

Returns

Action<object>

A delegate that calls the specified method, but accepts any type at compile time.

Remarks

This is relatively expensive as a one-time cost, but still significantly faster than Invoke(object, object[]) if used several times.
